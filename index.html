Project 4

Clock + API's App

I created a Clock App based on the Front-end Mentor’s project idea. I used their images, color scheme and I coded the website myself. I made HTTP Requests to public API’s: World Time API (https://worldtimeapi.org/api/ip), and Free Geo IP API (https://freegeoip.app/json/) to display the user’s local time/date and location. I also used a local JSON file that contains a list of quotes and a refresh quote button that will change the quote when clicked. I had difficulty storing the HTTP Request’s Response (user’s IP) from the World Time API, and then passing it to the next HTTP Request to Free Geo IP to get the user’s location. I solved this issue using a callback function to pass the response data from the first API call to the next to display the user’s location.

Project 5

Count Down Timer

I created a Countdown Timer based on the Front-end Mentor’s project idea. I used their images, color scheme and I coded the website myself. I called the Set Interval method which runs every millisecond and changes the Days, Hours, Minutes and Seconds accordingly. I created a variable that stores the current time using JavaScript’s new Date Object and another variable which stored the end of the countdown value using the Date Object set to December 31, 2021. I used the Get Time method to get each date’s value in Milliseconds. Then I used the Math Object and the Floor method to calculate how many Days, Hours, Minutes and Seconds are left in the year and displayed them in the document. I created a function that will run an animation each time one of the countdown timer clocks change.

Project 6

Ip Address Tracker

I created a IP Address tracker based on the Front-end Mentor’s project idea. I used their images, color scheme and I coded the website myself. The retrieves the User’s IP Address and the details about their ISP, location of the server and display the location on a map at the bottom of the page. The user can also type in a IP Address or Domain in the input field and the application will display details based on their input. I created an HTTP Request that ran when the window finished loading and retrieved the user’s IP Address and details about the IP provider and their location. I used the IP Geolocation API (https://geo.ipify.org/api/v1?apiKey= my api key) which required an API Key to display the data. An event listener takes the user’s input and creates a new API call. I ran a script in my head tag which was provided by leafletjs.com which displays your chosen coordinates and a location marker on a map of the world. I copied the free to use function they created to then display my user’s location or the location of the domain or IP Address they searched.
